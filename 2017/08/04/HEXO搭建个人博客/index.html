HEXO搭建个人博客

发表于2017-08-04字数统计123阅读时长1分
1. 使用NPM安装HEXO
npm install -g hexo
2. 初始化HEXO目录
hexo init
3. 生成静态页面
hexo generate 或 hexo g
4. 启动HEXO服务
hexo server
5. 安装HEXO主题
git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia
6. 创建GitHub仓库
仓库名格式：用户名.github.io
7. 修改HEXO部署配置
8. 部署到Github
npm install hexo-deployer-git --save
hexo deploy
9. 常用部署步骤
hexo clean
hexo generate
hexo deploy
